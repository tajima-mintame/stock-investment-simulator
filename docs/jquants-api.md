# J-Quants API V2 仕様・制限まとめ

## 1. 概要

J-Quants APIは日本取引所グループ（JPX）が提供する株式データAPI。本プロジェクトではV2を使用。

- 公式サイト: https://jpx-jquants.com/
- 認証方式: APIキー（`x-api-key` ヘッダー）
- Pythonクライアント: `jquantsapi` パッケージの `ClientV2` クラス

## 2. プランと制限

| プラン | レート制限 | データ履歴 | 月額 |
|--------|----------|----------|------|
| **Free** | 5リクエスト/分 | 2年（12週間遅延） | 無料 |
| Light | 60リクエスト/分 | 5年 | 有料 |
| Standard | 120リクエスト/分 | 10年 | 有料 |
| Premium | 500リクエスト/分 | 20年 | 有料 |

### 本プロジェクトで使用中のプラン: **Free**

- **レート制限: 5リクエスト/分**
- **データ範囲: 2024-01-21 〜 2026-01-21**（2年間、12週間遅延）
- レート制限超過時: HTTP 429（Too Many Requests）
- 連続超過時: **5分間のアクセスブロック**

### レート制限への対応（本プロジェクトの実装）

- APIコール間に `asyncio.sleep(1.0)` を挿入
- エラー発生時は追加で `asyncio.sleep(3.0)` 待機
- 1回のセットアップで20銘柄 × 2リクエスト（価格+財務）= 約40リクエスト → **約8分**必要

## 3. 本プロジェクトで使用しているエンドポイント

| メソッド | エンドポイント | 用途 |
|---------|-------------|------|
| `get_list()` | `/v2/equities/master` | 上場銘柄一覧（全4,400+銘柄） |
| `get_eq_bars_daily()` | `/v2/equities/bars/daily` | 日足OHLCV |
| `get_fin_summary()` | `/v2/financials/summary` | 財務サマリー（EPS, 売上高等） |

## 4. レスポンスのカラム名

### 上場銘柄一覧（`get_list()`）

| カラム | 説明 |
|--------|------|
| Code | 銘柄コード（5桁、末尾0。例: `72030` = トヨタ） |
| CoName | 企業名 |
| S33Nm | 33業種セクター名 |
| MktNm | 市場名（プライム/スタンダード/グロース） |
| ScaleCat | 規模区分（TOPIX Core30等） |

### 日足データ（`get_eq_bars_daily()`）

| カラム | 説明 |
|--------|------|
| Date | 日付 |
| Code | 銘柄コード（5桁） |
| O | 始値 (Open) |
| H | 高値 (High) |
| L | 安値 (Low) |
| C | 終値 (Close) |
| Vo | 出来高 (Volume) |
| AdjC | 調整後終値 |
| AdjFactor | 調整係数 |

### 財務サマリー（`get_fin_summary()`）

| カラム | 説明 | 用途 |
|--------|------|------|
| EPS | 1株当たり利益 | PER算出 |
| FEPS | 予想1株当たり利益 | 予想PER算出 |
| BPS | 1株当たり純資産 | PBR算出 |
| Sales | 売上高 | 成長率算出 |
| FSales | 予想売上高 | 成長率算出 |
| FDivAnn | 予想年間配当 | 配当利回り算出 |
| EqAR | 自己資本比率 | 財務健全性 |
| NP | 純利益 | 収益性 |

## 5. 銘柄コード変換

J-Quants V2 は5桁コード（末尾0付き）を使用する。本プロジェクトでは4桁で管理。

```
4桁 → 5桁: "7203" → "72030"
5桁 → 4桁: "72030" → "7203"
```

実装: `providers/jquants.py` の `_to_jquants_code()` / `_from_jquants_code()`

## 6. V1からV2への主な変更点

| 項目 | V1 | V2 |
|------|----|----|
| 認証 | リフレッシュトークン | APIキー |
| クライアント | `jquantsapi.Client` | `jquantsapi.ClientV2` |
| 銘柄一覧 | `get_listed_info()` | `get_list()` |
| 日足データ | `get_prices_daily_quotes()` | `get_eq_bars_daily()` |
| 企業名カラム | `CompanyName` | `CoName` |
| セクターカラム | `Sector33CodeName` | `S33Nm` |
| OHLCVカラム | `Open, High, Low, Close, Volume` | `O, H, L, C, Vo` |
| 調整後終値 | `AdjustmentClose` | `AdjC` |

## 7. Freeプランで利用可能なデータ

- 上場銘柄マスタ
- 株価（日足OHLCV）
- 財務データサマリー
- 決算カレンダー
- 取引カレンダー

### Freeプランで利用**不可**なデータ

- 分足・ティックデータ
- TOPIX指数
- 投資部門別売買状況
- 信用取引データ
- 空売りデータ
- 先物・オプションデータ
- 一括CSVダウンロード

## 8. 注意事項

- **12週間遅延**: Freeプランのデータは最新から12週間前まで。リアルタイムデータは取得不可
- **429エラー**: 5リクエスト/分を超えると発生。連続超過で5分間ブロック
- **データ範囲**: Freeプランは2年分のみ。範囲外の日付を指定するとHTTP 400
- **新規登録者**: 2025年12月22日以降の登録者はV2 APIのみ利用可能
